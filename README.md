# 📦 API Afrigaz Management

## 📌 Description

**API Afrigaz Management** est une API RESTful développée avec Laravel, conçue pour gérer efficacement le stock et les opérations liées aux produits d’une entreprise de distribution de gaz comme Afrigaz.

Ce système constitue le cœur backend permettant de centraliser, sécuriser et automatiser la gestion des stocks, des mouvements de produits et des opérations commerciales.

---

## 🎯 Objectifs du système

* 📊 Assurer une gestion précise et en temps réel des stocks
* 🔄 Suivre les entrées et sorties de produits (approvisionnement, ventes)
* 📦 Optimiser la gestion des bouteilles de gaz et accessoires
* 🧾 Faciliter la traçabilité des opérations
* 🔐 Fournir une API sécurisée pour les applications web et mobiles

---

## ⚙️ Technologies utilisées

* **Laravel** (Framework PHP)
* **Laravel API (RESTful)**
* **MySQL** (Base de données)
* **JWT / Sanctum** (Authentification API)
* **Postman** (Tests API)

---

## 🚀 Fonctionnalités principales

* 🔐 Authentification sécurisée (login / register)
* 📦 Gestion des produits (CRUD)
* 🏪 Gestion des stocks en temps réel
* 🔄 Gestion des entrées (approvisionnement)
* 📤 Gestion des sorties (ventes, distribution)
* 📈 Suivi des mouvements de stock
* 👥 Gestion des utilisateurs et rôles
* 🧾 Historique des opérations

---

## 🏗️ Architecture

L’API est construite selon une architecture REST respectant les bonnes pratiques :

* Controllers (logique métier)
* Models (Eloquent ORM)
* Routes API (`routes/api.php`)
* Middleware (sécurité et authentification)
